# 项目介绍
    本项目是北京邮电大学2024年软件工程（春）大作业——BUPT酒店空调管理系统，基于Django框架与mysql数据库开发。
 
# 环境依赖
    python 版本 3.10.8
    django 版本 4.2.2
    后面会添加Requirements.txt
    
# 使用说明
    先去安装redis
    然后进入项目根目录
    pip install -r requirements.txt
    redis-server
    python manage.py runserver
    python manage.py qcluster
    随后在127.0.0.1:8000可进行操作
